# Personal Portfolio

A professional online portfolio showcasing a résumé, projects, and a personal "scratch" page with interactive elements.

## Project Structure

- `src/pages/` - Main page components (Index, Resume, Contact, NotFound)
- `src/components/` - Reusable UI components
- `public/scratch/` - Interactive scratch page with custom styling

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
